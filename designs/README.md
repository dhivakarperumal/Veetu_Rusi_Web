Receipt design mockup

Files:
- receipt.html — mobile (1080x1920) receipt mockup (open in browser)
- receipt.css — stylesheet for the mockup

Preview:
- Open `designs/receipt.html` in your browser. For best results, open in a 9:16 viewport or set device toolbar in DevTools to 1080x1920.

Replace logo and DP photo:
- Edit `receipt.html` and change the `src` attribute of the logo or DP image to upload your own logo.

Export:
- Use browser Print → Save as PDF to create a print-ready receipt.
